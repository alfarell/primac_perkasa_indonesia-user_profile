# [PT Primac Perkasa Indonesia] Technical Test - User Profile with Vue3

Author: Alfarell Muchamad Yuwanto

A simple and elegant **User Profile Management** app built with **Vue 3**.  
The form automatically saves user data to `localStorage` after a short delay, no manual "Save" button required.

## Features

- Built with **Vue 3 Composition API**
- Styled using **PrimeVue** components and **TailwindCSS**
- **Auto-save** to localStorage with debounced
- Form state persists even after page reload
- Toast notifications for save status

## Tech Stack

- [Vue 3](https://vuejs.org/)
- [PrimeVue](https://primevue.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)

## Setup & Installation

### With npm

```bash
# Install dependencies

npm install
```

```bash
# Run development server

npm run dev
```

### With yarn

```bash
# Install dependencies

yarn
```

```bash
# Run development server

yarn dev
```
